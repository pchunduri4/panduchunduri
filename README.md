# panduchunduri

Personal site for **Pandu Chunduri** — AI product architecture, enterprise agentic systems, and AEO/GEO research. Built with [Next.js](https://nextjs.org/) 16 and React 19.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```
